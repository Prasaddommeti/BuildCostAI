# 🏗️ BuildCostAI

BuildCostAI is an AI-powered web application that helps users estimate construction costs quickly and accurately. It provides intelligent cost predictions, project planning support, and material cost analysis for residential and commercial buildings.

## 🚀 Features

- 🤖 AI-powered construction cost estimation
- 📊 Material cost analysis
- 🏠 Residential & commercial project support
- 🔐 Secure user authentication
- 📈 Interactive dashboard
- 💾 Save and manage project estimates
- 📱 Responsive and user-friendly interface

## 🛠️ Tech Stack

**Frontend**
- React
- Vite
- TypeScript
- Tailwind CSS

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose

**AI**
- Google Gemini API

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/Prasaddommeti/BuildCostAI.git
cd BuildCostAI
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
npm start
```

## 🔑 Environment Variables

Create a `.env` file in the `backend` folder:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key
```
