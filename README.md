# 🏗️ BuildCostAI

BuildCostAI is an AI-powered web application that helps users estimate construction costs for residential and commercial buildings.

## 🚀 Features

- AI-powered cost estimation
- User authentication
- Material cost analysis
- Project management
- Responsive design

## 🛠️ Tech Stack

- React
- Vite
- TypeScript
- Node.js
- Express.js
- MongoDB
- Gemini AI

## ⚙️ Installation

```bash
git clone https://github.com/Prasaddommeti/BuildCostAI.git
cd BuildCostAI
npm install
npm run dev
```
